# Tic Tac Toe Game

A C++ Tic Tac Toe game with 3x3 and 4x4 boards, supporting Player vs Player and Player vs Computer modes, a scoreboard, and a replay option.

## Language
C++

## Features
- 3x3 and 4x4 board
- Player vs Player mode
- Player vs Computer mode
- Scoreboard system
- Replay option

## How to Run
1. Open the project in **CodeBlocks**, **Dev C++**, or **Visual Studio**  
2. Compile the `main.cpp` file  
3. Run the program

## Rules
- Players take turns placing X or O on the board  
- First player to get 3 (or 4 for 4x4) in a row, column, or diagonal wins  
- Replay option allows starting a new game without closing the program
